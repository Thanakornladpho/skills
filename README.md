# Background Remover CLI

A simple yet powerful Python CLI tool to remove backgrounds from images using [rembg](https://github.com/danielgatis/rembg). It supports custom background colors and high-resolution images.

## Features

- **Automatic Background Removal**: Uses AI to detect and remove the background.
- **Custom Backgrounds**: Replace the background with any solid color (hex code or standard names like 'black', 'white').
- **High Resolution Support**: Optimized settings to handle large images (4K+) without crashing.
- **Transparent Output**: Saves as PNG with transparency coverage by default.
- **Manual Color Removal**: Remove specific contiguous background colors (e.g. solid black) by selecting them, avoiding AI misjudgment.
- **Watermark Removal**: Specifically designed to remove the "Generated by Google" watermark in the bottom-right corner.

## Installation

1.  **Clone or Download** this repository.
2.  **Create and Activate Virtual Environment**:
    It is recommended to use a virtual environment to avoid conflicts.

    **macOS / Linux:**

    ```bash
    # Check version first (must be < 3.14)
    python3 --version

    # If python3 is too new (e.g. 3.14), try using a specific version like 3.9, 3.10, or 3.11:
    # /usr/bin/python3 -m venv venv

    python3 -m venv venv
    source venv/bin/activate
    ```

    **Windows:**

    ```bash
    python -m venv venv
    venv\Scripts\activate
    ```

3.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
    _Note: The first run will download the U2NET model (approx. 100MB)._

## Usage

Run the script from your terminal:

```bash
python remove_bg.py [input_image] -o [output_image] [options]
```

### Arguments

- `input`: Path to the source image file.
- `-o, --output`: Path where the result should be saved.
- `-c, --color`: (Optional) Background color to apply.

### Examples

**1. Transparent Background (Default)**
Save as PNG to keep the transparency.

```bash
python remove_bg.py photo.jpg -o result.png
```

**2. White Background**
Great for e-commerce product photos.

```bash
python remove_bg.py product.jpg -o result.jpg --color white
```

**3. Black Background**

```bash
python remove_bg.py portrait.jpg -o result.jpg --color black
```

**4. Custom Color (Hex)**
Use quotes for hex codes.

```bash
python remove_bg.py logo.png -o new_logo.jpg --color "#FF5733"
```

### Batch Processing (Folder)

To process all images in a folder at once, simply point the input to the **folder**:

```bash
python remove_bg.py [input_folder] -o [output_folder] [options]
```

**Example:**

```bash
python remove_bg.py /path/to/photos -o /path/to/results --color white --model isnet-general-use
```

The script automatically detects if you provided a folder and switches to batch mode.

### Advanced Features

#### Alpha Matting (Better & Smoother Edges)

Use `-a` or `--alpha-matting` to enable post-processing that smoothens the edges.

```bash
python remove_bg.py photo.jpg -o result.png -a
```

You can fine-tune it with:

- `--af`: Foreground threshold (default 240)
- `--ab`: Background threshold (default 10)
- `--ae`: Erode size (default 10)

#### Model Selection

The default model is `u2net`. You can switch models for better results on specific images:

**General Use (Often sharper than u2net):**

```bash
python remove_bg.py photo.jpg -o result.png --model isnet-general-use
```

**Human Segmentation:**

```bash
python remove_bg.py person.jpg -o result.png --model u2net_human_seg
```

**Anime / High Detail:**

```bash
python remove_bg.py anime.jpg -o result.png --model isnet-anime
```

**Silhouette / Object Shape:**

```bash
python remove_bg.py object.jpg -o result.png --model silueta
```

_Note: The first time you use a new model, it will be downloaded automatically._

#### Post-Processing Mask (Cleaner Edges)

Use `-p` or `--post-process-mask` to enable additional cleanup of the mask edges. This can help with jagged lines but is slower.

```bash
python remove_bg.py photo.jpg -o result.png -p
```

### Manual Color Removal (Non-AI)

If the AI models are removing too much of your product (e.g., dark product on dark background), you can use the **Chroma Key** feature. This removes a specific contiguous color starting from the corners of the image.

**Arguments:**

- `-rc` or `--remove-color`: The color to remove (e.g., 'black', 'white', '#000000').
- `-t` or `--tolerance`: How strict the matching is (0-255). Lower is STRICTER (less removed), Higher is LOOSER (more removed). Default: 30.

**Example: Removing a solid black background:**

```bash
python remove_bg.py product.jpg -o result.png --remove-color black --tolerance 20
```

_Note: This uses a "Flood Fill" algorithm. It will only remove the background color connected to the corners, protecting the inner parts of your object even if they are the same color._

### Watermark Removal

To remove the standard AI watermark (e.g. from Google Gemini images) in the bottom-right corner:

```bash
python remove_bg.py image.png -o result.png --remove-watermark
```

This clears a 240x240 pixel area in the bottom-right corner.

## Troubleshooting

- **Memory Error**: If processing very large images (8K+), ensure your machine has enough free RAM. The script disables the default PIL pixel limit limit, but physical memory limits still apply.
- **First Run Slowness**: The first execution downloads the AI model. Subsequent runs will be much faster.
