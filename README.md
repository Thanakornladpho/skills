## 🖥️ What this app does

skills is a simple Windows app that helps you remove backgrounds from images. It is built as a Python command line tool, but this guide is written for everyday users who want a clear way to get it running.

You can use it to:

- Remove the full background from a photo
- Replace a background with a solid color
- Keep transparent edges in PNG files
- Work with large images
- Remove a solid color background by selecting that color
- Remove the “Generated by Google” watermark in the bottom-right corner

## 🚀 Getting started on Windows

### 1. Download the project

Open the download link above and get the project files from the repository page.

If you use Git, you can clone the project instead:

- Open Command Prompt
- Run:

git clone https://github.com/Thanakornladpho/skills

This creates a folder named skills on your computer.

### 2. Open the project folder

Find the folder named skills and open it.

You should see files and folders for the tool, along with a README file and Python files.

### 3. Install Python

If Python is not on your PC:

- Go to https://www.python.org/downloads/
- Download the latest Python 3 installer for Windows
- Run the installer
- Check the box that says Add Python to PATH
- Finish the setup

To check that Python works:

- Open Command Prompt
- Type:

python --version

If you see a version number, Python is ready.

### 4. Create a virtual environment

A virtual environment keeps this app separate from other Python tools on your PC.

In the project folder, open Command Prompt and run:

python -m venv venv

Then activate it:

venv\Scripts\activate

If it works, you will see (venv) at the start of the command line.

### 5. Install the required packages

Still in the same window, run:

pip install -r requirements.txt

This installs the tools the app needs to run.

If the project does not include a requirements file, install the main package directly:

pip install rembg

### 6. Run the app

To start the tool, use the main Python file in the project folder.

Try:

python main.py

If that file does not exist, look for another Python file that starts the app, such as:

- app.py
- cli.py
- remove_bg.py

Run it with:

python filename.py

## 🧭 How to use it

The app works from the command line. That means you type a command instead of clicking buttons.

A common use looks like this:

python main.py input.jpg output.png

This tells the app to:

- Read the file named input.jpg
- Remove the background
- Save the result as output.png

## 🎨 Common use cases

### Remove a background from a photo

Use a photo with one person or one object in front of a clear background.

The tool detects the subject and removes the rest.

Example:

python main.py photo.jpg photo-no-bg.png

### Replace the background with a color

You can set a solid background color. This helps when you need a white image for a shop listing or a black image for design work.

Example:

python main.py photo.jpg output.png --bg-color white

You can also use hex colors like:

- #ffffff
- #000000
- #ffcc00

### Keep a transparent background

If you want the image without any background, save it as PNG.

Example:

python main.py photo.jpg output.png

PNG keeps transparency, so the removed area stays clear.

### Remove a solid background color

If your image has one flat background color, such as black or green, the tool can target that color.

This helps when AI cutout results are not clean enough.

Example:

python main.py photo.jpg output.png --remove-color black

### Remove the “Generated by Google” watermark

The tool also supports removing a small watermark in the bottom-right corner.

Use it on files where the watermark sits in a fixed spot.

Example:

python main.py input.png output.png --remove-watermark

## 🔧 Troubleshooting

### Python command not found

If Windows does not know the python command:

- Reinstall Python
- Check Add Python to PATH
- Close Command Prompt and open it again

### The app does not start

Check that you are in the project folder before running the command.

You can move into the folder with:

cd path\to\skills

Then run the script again.

### Missing package errors

Run:

pip install -r requirements.txt

If that does not work, install the main package again:

pip install rembg

### Image file not found

Check:

- The file name
- The file type
- The folder location

If the file is on the Desktop, use the full path.

### Output looks wrong

Try a different image with:

- Clear lighting
- A plain background
- One main subject

For solid backgrounds, use the manual color removal option.
